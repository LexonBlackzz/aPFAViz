# ORIGINAL README BELOW:

So.. we have many many Black MIDI players for Android, but not one seemingly has tried to replicate PFAs engine...
UNTIL NOW! 
Over the few months or so, me and HexagonMIDIs have been working on something called "aPFA" or "Android PFA" that intends to be a direct replication of PFAs engine to Android! It's in a state to where I'm confident enough in saying that you can legit run on your phone in good faith!

aPFAs current features as of **v1.2.0** are:
- Manual/Automatic core affinity chooser. (can choose various cores, or if you don't care, aPFA will choose your strongest core!) 
- Track skimming (if a little slow, aka you can go to random points in the MIDI) 
- A near 1:1 of PianoFromAboves actual engine! 
- Background color chooser along with custom images
- Color profiles!
- Legacy OpenGL ES 2.0 renderer
- **Pagefiling! (and chunked paging!) for very large MIDIs!**
